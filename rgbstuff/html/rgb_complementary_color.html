What's a complementary color ?

Its the color that is opposite to the color you choose.
It allows you to get a color that will be in contrast (the most noticeable) with the one you choose.

How it's calculated ?

We take the R,G and B of our color and make this calculation :

255 - R, 255 - G, 255 - B

This allows us to get the values for the complementary color !

Example

  • 255 - 255 = 255 | 255 - 255 = 255 | 255 - 255 = 255
  • RGB : 255, 255, 255
